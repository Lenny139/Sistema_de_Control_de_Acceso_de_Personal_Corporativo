1. Descripción General

El diseño del Sistema de Control de Acceso está centrado en la usabilidad, eficiencia y experiencia del usuario. Cada página está diseñada para facilitar la tarea específica de cada rol de usuario.

3. Tipografía

Fuente Principal

Familia: Segoe UI, Roboto, sans-serif

Pesos disponibles: 400 (Regular), 600 (Semibold), 700 (Bold)

Escalas de Tipografía

Heading 1 - 2.5rem / 700

Usado en: Encabezados de página principal

Heading 2 - 2rem / 700

Usado en: Títulos de sección

Heading 3 - 1.5rem / 700

Usado en: Subtítulos

Body Text - 1rem / 400

Usado en: Texto de párrafos principal

Small Text - 0.875rem / 400

Usado en: Texto secundario, labels

Recomendaciones

  • Mantener línea base de 1.6 entre líneas
  • Máximo 80 caracteres por línea en desktop
  • Usar bold solo para destacar información importante
  • No exceder 3 niveles de jerarquía en una página

5. Sombras

Niveles de Sombra

Shadow SM
Shadow MD
Shadow LG
Shadow XL

Especificaciones

--shadow-sm: 0 1px 2px 0 rgba(0, 0, 0, 0.05); --shadow-md: 0 4px 6px -1px rgba(0, 0, 0, 0.1); --shadow-lg: 0 10px 15px -3px rgba(0, 0, 0, 0.1); --shadow-xl: 0 20px 25px -5px rgba(0, 0, 0, 0.1);

Uso

  • Shadow SM: Form inputs, pequeños elementos
  • Shadow MD: Cards, modales pequeños
  • Shadow LG: Modales, hover states
  • Shadow XL: Overlays, elementos destacados

9. Responsividad

Media Queries

/* Tablet */ @media (max-width: 768px) { .sidebar-layout { grid-template-columns: 1fr; } .card { padding: 1rem; } } /* Mobile */ @media (max-width: 520px) { .header { flex-direction: column; } .kpi-grid { grid-template-columns: 1fr; } }

Consideraciones Mobile

  • Mínimo 44x44px para elementos interactivos
  • Font size mínimo 16px en inputs
  • Touch area padding: 8px
  • Sidebar debe colapsar o ser drawer
  • Tablas en mobile pueden horizontales o convertirse a cards
